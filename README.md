## 🧱 Cấu trúc dự án

```bash
project-management-frontend/
│
├── index.html              # Layout chính
│
├── pages/                 # Trang con (load động)
│   ├── dashboard.html
│   ├── projects.html
│   ├── tasks.html
│
├── css/                   # Styles
│   ├── main.css
│   ├── layout.css
│   ├── components.css
│
├── js/                    # Logic
│   ├── app.js             # Router SPA
│   ├── api.js             # API handler
│   ├── project.js         # Logic dự án
│   ├── task.js            # Logic công việc
│   ├── utils.js           # Helper functions
│
├── data/
│   └── mock-data.json     # Dữ liệu test
```

## ⚙️ Cài đặt & chạy

### 1. Clone project

```bash
git clone https://github.com/phamvana/construction-project-management-frontend.git
```

### 2. Mở project

```bash
cd construction-project-management-frontend
```

### 3. Chạy

👉 Mở file `index.html` bằng trình duyệt
hoặc dùng Live Server (VS Code)

---

## 🌐 Deploy GitHub Pages

1. Vào repo → **Settings**
2. Chọn **Pages**
3. Source: `main` branch
4. Save

👉 Truy cập:

```
https://phamvana.github.io/construction-project-management-frontend/
```
